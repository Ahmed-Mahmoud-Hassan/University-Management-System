# University Management System

A comprehensive web-based University Management System that provides separate portals for students, professors, and administrators.

## Features

### Student Portal
- View and manage courses
- Access grades and academic progress
- View schedule and assignments
- Communication with professors

### Professor Portal
- Manage courses and students
- Post assignments and grades
- Schedule management
- Communication with students

### Admin Portal
- User management
- Course management
- Department management
- System configuration

## Technology Stack
- HTML5
- CSS3
- JavaScript
- Authentication System

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/Joe-world1357/University-Management-System.git
```

2. Open the desired portal in your web browser:
   - `login-advanced.html` - Main login page
   - `student-portal.html` - Student interface
   - `professor-portal.html` - Professor interface
   - `admin-portal.html` - Administrator interface
